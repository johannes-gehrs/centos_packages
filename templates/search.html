Search CentOS packages online

centos-packages.com

  • Toggle search help
  • All packages
  • Searching for more than one word creates an implicit AND connection.
  • Quoting "multiple words" searches for exact occurences of "multiple words".
  • Boolean operators "NOT", "AND" and "OR" are supported. You can use parentheses to create groups.
  • You can use unix style wildcards. "*" matches any characters, "?" matches single characters.
  • Package information is structured in fields. By default the following fields are searched: "name", "summary", and "description".
  • You can alternatively search for specific fields, like so: "description: kernel".
  • The following fields are also available: "arch" (CPU architecture), "version", "release", "url" (vendor site), "license".
